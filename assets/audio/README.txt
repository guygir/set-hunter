Set Hunter — Audio Tracks
=======================
Place MP3 files here for the 🎵 Music Tracks mode.
Enable it in-game with the 🎵 button in the HUD.

Required files:
  normal.mp3   — calm background loop
  tense.mp3    — low-budget / late-game tension
  hype.mp3     — pack-opening excitement
  victory.mp3  — run completion

Suno.com prompts (free tier — 50 credits/day, each song ~5 credits):
----------------------------------------------------------------------
normal.mp3:
  lo-fi chill, 75 BPM, soft piano, light synth pads, no lyrics,
  seamless loop, relaxing card-game background music, warm and cozy

tense.mp3:
  dark ambient, minor key, 90 BPM, low strings, tension drone,
  no lyrics, seamless loop, trading card game suspense, ominous

hype.mp3:
  upbeat electronic, 120 BPM, synth pop, triumphant brass stabs,
  no lyrics, seamless loop, pack opening excitement, energetic

victory.mp3:
  cheerful jazz, Cmaj, bright piano and pizzicato strings,
  triumphant fanfare, no lyrics, 8-second seamless loop,
  game-completion celebration

How to generate on Suno:
  1. Go to https://suno.com  →  Create (free account)
  2. Paste a prompt above, enable "Instrumental", click Create
  3. Download the MP3  →  save here as the filename above
